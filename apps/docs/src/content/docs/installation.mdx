# Installation

## Requirements

- Node.js 18 or higher
- React 18 or higher
- Next.js 13 or higher (App Router supported)

## Package Manager

Choose your preferred package manager:

### npm

```bash
npm install next-unified-query
```

### pnpm

```bash
pnpm add next-unified-query
```

### Yarn

```bash
yarn add next-unified-query
```

## TypeScript Support

The library is written in TypeScript and provides full type safety out of the box. No additional setup is required.

## Verification

After installation, verify that the package is working correctly:

```typescript
import { QueryClient } from 'next-unified-query';

// Create a new query client
const queryClient = new QueryClient();

console.log('next-unified-query is ready!');
```
